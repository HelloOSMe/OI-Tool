# OI-Tool

欢迎使用OI-Tool。

使用方法：
把文件放到C盘随便哪个文件夹下吃灰，然后在C:/WINDOWS建立一个快捷方式链接到文件。

好了，试着在cmd里输入oi-tool吧！
